Place Arabic-capable .ttf font files in this folder.

Recommended filenames:
- NotoNaskhArabic-Regular.ttf
- NotoNaskhArabic-Bold.ttf
- NotoSansArabic-Regular.ttf
- NotoSansArabic-Bold.ttf
- Amiri-Regular.ttf
- Amiri-Bold.ttf

The PDF generator will try this folder automatically before system font paths.
